Class transformer_phpsource

Description
Description | Vars (details) | Methods (details)

Highlights php code where specified by the 'match' attribute The default value for that attribute is "//code[@language='php']", which is what I use for Slideml processing.

This code is for example doing the highlighting on http://edu.mediagonal.ch/php/php5-lucerne/slide_21.html

Note that you have to use <![CDATA[ ..code.. ]]> in case you have offending strings such as "&" or "<?php" in your code.

Security implications: none I know of
