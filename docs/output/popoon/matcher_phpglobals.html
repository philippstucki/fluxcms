Matches one of PHPs super global variables

(like $_SERVER, $_GET, etc...)

Attributes: var: name of the var key: name of the to be matched key (eg. REQUEST_URI)

Example: <map:match type="phpglobals" var="_SERVER[REQUEST_URI]" pattern="*">
