Generator Module for popoon

This Module provides access to the popoon , with its structure Files and sql2xml stuff and some config options in the db, etc...

It should be a little bit cleaned up. It grew over the last few months and therefore it's a mess :) it has some special features as well, which were only needed on some sites, therefore we need something more modular here

It extends the structure2xml module for the main computing part

It's a big mess, should be documentated and cleaned up some day...
