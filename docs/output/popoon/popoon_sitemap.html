		generateSitemapID (line 286)
	
 
	
	

generates a unique ID out of a string (path+filename)


This method is used for generating a unique ID for every sitemap   which has to be compiled. It just takes the realpath as input,   and returns a unique ID for it.  In this case it replaces every DIRECTORY_SEPERATOR with _ for better   debugging. Theoretically one could use md5 as well.
