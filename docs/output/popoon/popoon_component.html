Check validity of a validityObject from cache: NOT REALLY DONE YET!!


It's the same thing as with all validity checkers: there are many more ways to determine  cache validity. things like "access + 4 days" (which would also involve sending appropriate http headers)  are not implemented at all. but they will be! .)
